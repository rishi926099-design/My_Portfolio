## ⚙️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/rishi926099-design/My_Portfolio.git
```

### Install Dependencies

#### Frontend

```bash
cd Frontend
npm install
npm run dev
```

#### Backend

```bash
cd Backend
npm install
npm start
```
